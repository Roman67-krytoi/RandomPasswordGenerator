# Random Password Generator

**Автор:** Роман Барышников

## Описание

Приложение с графическим интерфейсом (GUI) для генерации случайных паролей с возможностью настройки параметров и сохранения истории.

## Возможности

- Настройка длины пароля (4–32 символа) с помощью ползунка
- Выбор типов символов: цифры, буквы, спецсимволы
- Генерация случайного пароля одним нажатием
- Копирование пароля в буфер обмена
- Сохранение истории паролей в JSON
- Загрузка истории при запуске
- Очистка истории

## Требования

- Python 3.6+
- Стандартные библиотеки: `tkinter`, `random`, `string`, `json`, `os`, `datetime`
